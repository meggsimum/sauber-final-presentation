SAUBER Abschlusspräsentation, 09.12.2021

Gesamtsystem

SAUBER

ein schnittstellenbasiertes System
  • Service-orientierte Architektur
  • Geodateninfrastruktur als Basis aller SAUBER Dienste
  • Automatisierte Veröffentlichung von neuen Daten
  • offene Standards
  • Demonstratoren für die Anbindung an standardisierte Geo-Services

SAUBER Plattform

GDI - Geodateninfrastruktur

  • voll automatisiert
  • flexibel durch Microservices
  • erweiterbar auf mehr Regionen und Schadstoffe
  • skalierbar auf größere Last

Schnittstellen

  • OGC-Standards: WMS, WFS, WCS
  • räumliche Filter
  • zeitliche Filter
  • nutzbar von vielen Clients (Browser, Desktop ...)
